How it works

Select or type → done

1

Trigger

Shortcut, floating icon, or auto on select.

2

Capture

Reads the selection via Accessibility, with clipboard fallback.

3

Translate

Streams from your chosen local or cloud model.

4

Show or replace

Popup to read, or paste in place to write.

Your model, your call

Local or cloud

🖥️ Local · Ollama

Offline and private. Install a model and go — Saypick auto-picks an installed one.

ollama pull qwen2.5:3b

☁️ OpenAI-compatible

Point it at any /chat/completions endpoint — official API, a proxy, or your own server. Base URL + key + model.
